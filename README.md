# user-hotdeal-bot

한국 커뮤니티 유저 핫딜 알림 봇

## 주요 기능
한국 커뮤니티들의 유저 핫딜 게시판을 크롤링하여 텔레그램 등의 봇으로 알림

- 핫딜 종료, 제목 수정, 게시글 삭제 등 상태 변화 대응
- 나름 유연한 구조로 만들어 그럭저럭 괜찮은 확장성
- 열심히 배워서 그럴듯하게 적용한 로깅
- 기능은 아니지만, 추천인 광고 없을 예정
